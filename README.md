# UrHouseBot
save Ur time, Ur patient, to find a appropriate house to Rent
方便的挂机搜房, 只是一个很简单的解放眼睛的监控需求(目前), 因为搜房子太累了啊有木有...

# 当前思路:
-> 通过豆瓣的搜索小组搜索关键字找出对应的小组
-> 看小组里的话题新鲜度. 
-> 进入小组话题里搜索相关的关键字 
-> 进入相关的帖子里 
-> 解析出对应的街道关键字/房租/房屋情况 等信息(因为帖子中没有固定的格式, 所以需要用正则匹配有可能的关键字, 筛选出来的一起打印, 由个人人工筛选)
